Description

Cube Adventure is a simple borderless mini-adventure game where players control a character using the keyboard, navigate around randomly generated mountains, and reach randomly placed goal points to score.

Technical

- Added a player entity controlled by WASD keys.

- Used XXH.h32 hash values to generate reproducible and randomized map features (mountains, treasures).

- Implemented a goal point system where players gain points upon reaching new randomly placed targets.

- Introduced moveDelay timing to regulate movement speed, making gameplay more strategic.

- Developed a real-time distance feedback system showing how far the player is from the current target.

Reflection

This project was the most game-like and involved more complex logic, including movement, collision, and scoring. I gained a deeper understanding of how to integrate procedural world generation with gameplay rhythm and feedback systems. One of the challenges was managing overly fast movement, which I solved by introducing a movement delay, resulting in a much smoother experience.
