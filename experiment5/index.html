Description

This project builds an evolutionary image generator that takes inspiration from iconic images and attempts to recreate their visual impression through iterative mutation and evaluation.

Technical

Using p5.js, I implemented a generator that starts with a randomly initialized design and mutates it over time based on a Gaussian function. Each element (box or shape) samples color directly from the source image, and every mutation changes not only position and dimensions but also updates color based on the new coordinates.
The mutation rate can be adjusted via a slider, allowing the user to control the pace of evolution. I also added a shape selector dropdown that supports rectangles, ellipses, and triangles. The system scores each generated design using a pixel-wise comparison with the inspiration image and preserves the highest-scoring result.
