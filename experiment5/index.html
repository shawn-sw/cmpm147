Description

This project aims to use procedural generation techniques to build two tile-based worlds, an overworld and a dungeon.

Technical

Use p5.js and the provided tileset to create a randomly generated environment. Using functions like noise, I get the output I want. Adding elements ensures that the landscape is randomly generated every time the user presses the "reseed" button.

Reflection

This experiment was by far the hardest one for me. I had never used tilesets in p5.js before, and the main difficulties for me were corner tiles, animated tiles, and the use of tilesets. Also, generating a second map with a different structure was a challenge. I consulted with some friends to get my map to work properly.
